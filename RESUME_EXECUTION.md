# 스크립트 재실행 가이드

## 중단 후 재실행

스크립트를 중간에 중단해도 **안전하게 재실행**할 수 있습니다!

### 개선된 기능

1. **자동 건너뛰기**
   - 이미 처리된 파일은 자동으로 건너뜀
   - 파일 수정 시간을 비교하여 최신 파일 유지

2. **안전한 재실행**
   - 중단 후 다시 실행해도 문제없음
   - 이미 이동된 파일은 다시 처리하지 않음

3. **진행 상황 표시**
   - 100개마다 진행 상황 출력
   - 처리 결과 요약 제공

## 재실행 방법

### 내일 다시 실행할 때

```bash
python organize_html_to_folders.py
```

### 동작 방식

1. **이미 처리된 파일**
   - 타겟 파일이 이미 존재하고 최신이면 건너뜀
   - "이미 처리됨" 메시지 표시

2. **새로 처리할 파일**
   - 아직 처리되지 않은 파일만 이동
   - 경로 수정 및 업데이트

3. **업데이트된 파일**
   - 소스 파일이 더 최신이면 업데이트
   - 기존 파일은 백업

## 예상 결과

### 첫 실행
```
✅ 새로 이동: 2500개
✓ 이미 존재: 0개
⏭️ 건너뜀: 50개
```

### 재실행 (중단 후)
```
✅ 새로 이동: 500개 (나머지)
✓ 이미 존재: 2000개 (이미 처리됨)
⏭️ 건너뜀: 50개
```

## 주의사항

1. **파일이 많으면 시간이 걸립니다**
   - 2500개 파일 처리 시 약 10-30분 소요 가능
   - 중단해도 안전하게 재실행 가능

2. **진행 상황 확인**
   - 100개마다 진행 상황 출력
   - 터미널을 닫아도 재실행 가능

3. **완료 확인**
   - 모든 파일이 처리되면 "모든 작업 완료!" 메시지
   - 처리 결과 요약 확인

## 문제 해결

### 재실행 시 오류가 발생하면

1. **파일 권한 확인**
   - 파일이 다른 프로세스에서 사용 중인지 확인

2. **디스크 공간 확인**
   - 충분한 디스크 공간이 있는지 확인

3. **로그 확인**
   - 오류 메시지를 확인하여 문제 파악
